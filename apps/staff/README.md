# fam2026/staff

Internal application for managing reservation, booking, and participant data for CPE Family 2026

## Development Setup


Create Admin account using `bun run create-admin <email> <username> <password> [name]`

Example:

```bash
bun run create-admin anyemail@example.com username password example name
```

6. Go to `http://localhost:3037` and login with credentials you just created.
